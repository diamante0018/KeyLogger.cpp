# KeyLogger.cpp
Contains source code for a keylogger in c++, work on Windows
This repository contains the code for two applications: keylogger.cpp and spawnkeylogger.cpp
keylogger.cpp is the main application that registers the keys typed bu the user, spawnkeylogger.cpp is needed in order to create a background 
process of keylogger.exe so that it stays hidden from the user.
